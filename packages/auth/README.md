# @tedx-2026/auth

Authentication and session management using better-auth.

## Purpose

Provides authentication configuration and session management for the application.

## Key Responsibilities

- Configure better-auth
- Manage user sessions
- Provide auth helpers and types
- Handle login/logout operations

## Main Exports

- `createAuth` - Auth instance factory
- `Session` - Session type
- `User` - User type
- Auth middleware

## When to Use

- Implementing login/logout
- Checking user sessions
- Protecting admin routes
