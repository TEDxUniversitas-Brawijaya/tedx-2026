# @tedx-2026/ui

Shared React components and design system based on shadcn/ui.

## Purpose

Provides reusable UI components and implements the design system from Figma.

## Key Responsibilities

- Provide reusable UI components (Button, Input, Card, etc.)
- Implement design system from Figma
- Ensure design consistency between store and dashboard
- shadcn/ui components with Tailwind CSS

## Main Exports

- Button, Input, Card (shadcn/ui components)
- Custom components
- Theme configuration

## When to Use

- Building UI for store or dashboard apps
- Ensuring design consistency
- Reusing common UI components

- [Architecture](../../docs/architecture.md)

## Note

Dashboard should use components as-is. Keep it simple, clean, and consistent.
